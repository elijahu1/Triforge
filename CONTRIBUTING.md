# Contributing to Triforge

Thanks for your interest. Here's how it works:

## Rules
- Fork the repo, make your changes, open a PR
- Don't break the IaC layer separation — Pulumi owns SageMaker, Terraform owns EC2, OpenTofu owns S3
- Document what you changed and why
- No PRs that add dependencies without a good reason

## What's welcome
- Bug fixes
- Better nginx configs
- Improved Flask proxy
- New model support
- Documentation improvements

## What's not welcome
- Merging all 3 IaC tools into one — that defeats the purpose
- Adding paid services without flagging cost implications
